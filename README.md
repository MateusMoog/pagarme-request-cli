# Pagar.me Request CLI

A simple CLI http requester to verify the response from local pagarme API.

## Developing

1. **Clone this repo**
```sh
$ git clone git@github.com:mateusmoog/pagarme-request-cli
```

2. **Install dependencies**
```sh
$ yarn
```
OR
```sh
$ npm init
```

3. **Have fun**
